# CookBook - Кулинарная книга

## Описание

CookBook - это iOS приложение-кулинарная книга, разработанное на SwiftUI с использованием Core Data для хранения данных. Приложение позволяет пользователям создавать, сохранять и просматривать рецепты блюд.

## Технологии

- **SwiftUI** - современный фреймворк для создания пользовательского интерфейса
- **Core Data** - фреймворк для управления данными и их персистентности
- **Xcode** - среда разработки

## Структура проекта

```
CookBook/
├── CookBook/
│   ├── CookBookApp.swift          # Главный файл приложения
│   ├── ContentView.swift          # Основное представление
│   ├── Persistence.swift          # Настройки Core Data
│   ├── CookBook.xcdatamodeld/     # Модель данных Core Data
│   └── Assets.xcassets/           # Ресурсы приложения
├── CookBookTests/                 # Модульные тесты
└── CookBookUITests/               # UI тесты
```

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `CookBook.xcodeproj` в Xcode
3. Выберите симулятор или подключите устройство
4. Нажмите Run (⌘+R)

## Требования

- iOS 14.0+
- Xcode 12.0+
- Swift 5.3+
